# Lab 2
## ADS Problem Statement
Office Manager can
- Register Dentist
- Enroll new patients
- Confirm patients’ appointment
- Sign in/out

Dentist have
- Unique ID
- First Name
- Last Name
- Contact Phone Number
- Email
- Specialization

Dentist can
- View all their appointments
- View their patient’s details
- Sign in/out

Dentist can’t
- Get more than 5 appointments per week

Patient have
- FirstName
- Last Name
- Contact Phone Number
- Email
- Mailing Address
- Date Of Birth
